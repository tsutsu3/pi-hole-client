## What you can do on this screen

The About section provides **reference information about the app itself**.
Nothing on this screen changes Pi-hole behavior or server configuration.

Use this area to:
- Check the app version and support links
- Review privacy settings and diagnostics options
- Read legal notices and open source licenses

This section is safe to browse at any time.

---

## Application details

The **Application Detail** page shows basic information about the app,
including version and build details.

<img width="320" alt="Application detail screen" src={imgAboutMain} />

Open this page when:
- Reporting a bug or requesting support
- Checking whether you are running the latest version
- Verifying build or environment information

Support links open in your browser and do not affect the app state.

## Privacy and diagnostics

The **Privacy** page explains how the app handles data and diagnostics.

<img width="320" alt="Privacy settings screen" src={imgAboutPrivacy} />

On this page, you can:
- Read the privacy policy
- Enable or disable crash report sharing

Crash reports help improve the app by sending anonymized diagnostics.
Enable this option only if you agree to share this information.

Some changes may require restarting the app to take effect.
